# Rick and Morty Character List

## Getting Started

1. Clone the repository:

```
$ git clone https://github.com/allfitra/mostrans-characters-test.git
```

2. Install dependencies:

```
$ npm install
```

3. Run the development server:

```
$ npm run dev
```

4. Explore, contribute, and enjoy!
